「カメコロ」（以下「本アプリ」）は、個人開発者 7ofu（以下「開発者」）が提供する
爬虫類の給餌パズルゲームです。本ポリシーは、本アプリが取得する情報とその取り扱いに
ついて説明します。

## 1. 収集する情報と利用目的

本アプリが取得する情報は以下のとおりです。実名・メールアドレス・電話番号などの
個人情報の入力は求めません。

| 情報 | 内容 | 利用目的 |
|---|---|---|
| 匿名認証 ID（uid） | Firebase Anonymous Authentication が生成する、個人を特定しない機械的な識別子 | ランキング機能で「あなたの記録」を判別するため |
| スコア | プレイ結果の得点 | 歴代・日次ランキングの表示（**全ユーザーに公開されます**） |
| ニックネーム | 利用者が任意入力する表示名（1〜12文字） | ランキング上での表示（**全ユーザーに公開されます**） |
| 日付（day） | スコア送信日（YYYY-MM-DD） | 日次ランキングの集計 |
| 更新日時（updatedAt） | スコアを記録したサーバー側の日時（サーバー自動付与） | ランキングの更新日時の記録 |
| クラッシュ・診断データ | アプリの異常終了時のスタックトレース等（Firebase Crashlytics） | 不具合の把握・修正（**リリースビルドのみ収集し、個人を特定する情報は含めません**） |
| デバイス識別子（Firebase Installation ID） | Firebase Crashlytics が診断データに紐づけるため自動的に発行するインストール単位の識別子（開発者が明示的に収集するものではありません） | クラッシュ・診断データの集計（**リリースビルドのみ**） |

上記のうち、匿名認証 ID・スコア・ニックネーム・日付・更新日時は、ランキング機能を
利用した場合にのみ送信されます。オフラインのまま遊ぶ場合、これらの情報は端末外へ
送信されません。

### ニックネームについて（重要）

ニックネームはランキング上で**全ユーザーに公開表示**されます。実名や連絡先など、
公開されると困る情報を入力しないでください。不適切な語句・URL・制御文字は
アプリ内フィルタで拒否されますが、フィルタをすり抜けた表示や、他の利用者による
不適切な表示名を見つけた場合は、下記「7. お問い合わせ・削除請求」の連絡先まで
ご連絡ください。

## 2. 収集していない情報

本アプリは以下の情報を**取得しません**。

- 実名・住所・電話番号・メールアドレス
- 位置情報
- 連絡先（アドレス帳）
- 広告識別子（IDFA / Google 広告 ID）
- 上記を用いたトラッキング・行動ターゲティング広告

本アプリに広告は表示されません（ADR-0013）。

## 3. 端末内にのみ保存する情報（送信しません）

以下の情報は端末内（shared_preferences）にのみ保存され、開発者のサーバーや
第三者へ送信されることはありません。

- 自己ベストスコア（TOP3）
- 設定（ニックネームの下書き・言語・スキン・BGM/効果音のON/OFF）
- 図鑑（コレクション）の記録
- アプリ内課金（IAP）の解放状態のキャッシュ（真実の情報は Apple/Google の
  決済プラットフォームが保持し、本アプリはオフライン時の表示ちらつき防止の
  ためだけにキャッシュします）

アプリを削除すると、これらの情報は端末から消去されます。

## 4. 第三者への提供（データ処理の委託）

本アプリは、以下の Google（Firebase）のサービスを利用して 1. の情報を処理・
保存しています。これは第三者への「販売」や「共有」ではなく、本アプリの機能を
実現するための業務委託（データ処理の委託）です。

- **Firebase Authentication**（匿名認証 ID の発行）
- **Cloud Firestore**（スコア・ニックネームの保存、ランキングの集計・表示）
- **Firebase Crashlytics**（クラッシュ・診断データの収集）
- **Firebase App Check**（不正なリクエストの遮断。追加の個人情報は取得しません）

これらのサービスの提供者である Google の個人情報の取り扱いについては、
[Google プライバシーポリシー](https://policies.google.com/privacy) をご確認
ください。

アプリ内課金（IAP）をご利用の場合、決済処理は Apple App Store / Google Play が
行い、本アプリはお客様のお支払い情報（クレジットカード番号等）を一切取得・
保存しません。

上記以外の第三者への情報提供は行いません。

## 5. データの保存先

スコア・ニックネーム等は Google Cloud Platform 上の Cloud Firestore
（Firebase）に保存されます。データセンターの所在地は Google の運用方針に
準じます。

## 6. お子様のプライバシー（保護者の方へ）

本アプリは年齢レーティング 4+（全年齢向け）で提供しており、お子様にもご利用
いただけます。本アプリは実名・メールアドレス・位置情報など、児童から個人を
特定できる情報を収集しません。

ただし、ランキング機能のニックネームは他の利用者に公開表示される仕組みで
あるため、**お子様がニックネームを設定する際は、実名や個人が特定できる文字列を
入力しないよう、保護者の方が確認・指導してくださるようお願いいたします**。
不適切な語句・URL は自動フィルタで拒否されますが、フィルタは完全ではありません。
公開表示に不安がある場合は、ランキング機能を使わずオフラインのままでも
本アプリの主要な機能（パズルのプレイ・自己ベストの記録・図鑑）をすべて
お楽しみいただけます。

米国の児童オンラインプライバシー保護法（COPPA）等、適用される各国の子供向け
プライバシー保護規制の趣旨を踏まえ、本アプリは児童を対象とした行動ターゲティング
広告・第三者トラッキングを一切行いません（そもそも広告自体を表示しません）。

関する通報、データの削除請求は、以下のメールアドレスまでご連絡ください。

**連絡先: 7ofu.dev@gmail.com**

### データ削除について（正直な説明）

本アプリはメールアドレス等の連絡先情報を保持しないため、匿名認証 ID（uid）だけを
手がかりに「どの記録がどなたのものか」を技術的に特定することが困難です。この
ため、**個々のスコア・ニックネームの記録を、利用者ご本人からのお申し出だけで
確実かつ迅速に特定・削除することはお約束できません**。

それでも、上記の連絡先までご連絡いただければ、可能な範囲で対応いたします。
たとえば、直近のプレイ日時・記録したスコア・ニックネーム等、記録の特定に
役立つ情報をお知らせいただくことで、該当する記録を Firebase の管理コンソール
から手動で削除できる場合があります。ランキング全体からご自身のすべての記録を
確実に削除したい場合は、その旨を明記のうえご連絡ください。

なお、アプリを削除するだけでは、既にランキングへ送信済みのスコア・ニックネーム
は自動的には削除されません（3. に記載の端末内データのみ消去されます）。
